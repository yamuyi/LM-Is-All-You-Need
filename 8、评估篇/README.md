# 大模型评估

# 评估篇：大模型评估体系与实践

## 目录说明

本篇聚焦大模型评估的核心维度、主流方法及实战工具，涵盖从基础性能到复杂能力的全方位评估方案。

## 学习目标

1. 掌握大模型评估的核心维度（通用能力/专业领域/安全性等）
2. 能独立设计评估方案并使用主流工具（如MT-Bench、CEval）开展测评
3. 理解人工评估与自动评估的适用场景及协同方式
4. 学会分析评估结果并定位模型优化方向

## 核心内容

| 子目录/文件              | 说明                                   |
| ------------------------ | -------------------------------------- |
| 01_Evaluation_Dimensions | 评估核心维度（语言理解/推理/生成等）   |
| 02_Automatic_Evaluation  | 自动评估工具实战（MT-Bench/CEval等）   |
| 03_Human_Evaluation      | 人工评估方案设计（评分标准/流程规范）  |
| 04_Safety_Evaluation     | 安全性与对齐评估（偏见/ toxicity检测） |
| 05_Evaluation_Analysis   | 评估结果分析与模型优化建议             |

## 资源推荐

* **工具** ：Hugging Face Evaluate、FastEval（快速评估框架）、Eleuther AI LM-Harness
* **基准** ：[MT-Bench](https://github.com/lm-sys/FastChat)、[CEval](https://github.com/THUDM/CEval)、[MMLU](https://github.com/hendrycks/test)
* **论文** ：[Holistic Evaluation of Language Models](https://arxiv.org/abs/2211.09110)、[Truthful AI](https://arxiv.org/abs/2109.07958)
