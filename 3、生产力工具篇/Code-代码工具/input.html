转行大模型，新人该怎么选方向？少走弯路的经验谈

结合我个人在大模型领域的经验，聊聊新人如何转行大模型赛道，帮你少踩坑、找准路。

如果你正在准备投校招或社招，想进入大模型这个热门领域，那这篇文章应该能帮到你。咱们不绕弯子，直接上干货。

01 大模型都有哪些方向？

你要是去招聘网站搜"大模型"，看一圈 JD 就会发现，岗位大致可以分为四类：

1. 做数据的

  • 大模型数据工程师
  • 负责爬虫、数据清洗、ETL、Data Engine、Pipeline 等
  • 核心是搞"高质量数据"

2. 做平台的

  • 大模型平台工程师
  • 搞分布式训练、大模型集群、工程基建
  • 目标是让训练更快、更稳

3. 做应用的

  • 大模型算法工程师
  • 比如搜索、广告、推荐、对话机器人、AIGC
  • 直接面对业务，做模型调优和应用落地

4. 做部署的

  • 大模型部署工程师
  • 负责推理加速、跨平台部署、端侧智能、嵌入式
  • 目标是把模型跑得更省、更快

02 大部分新手的误区

看到这四个方向，你是不是第一反应就想选"做应用的"？

很多人都觉得做算法最前沿、最核心，能做出看得见的产品。但我想泼点冷水：做应用很吃业务经验。

如果你本来就是做 NLP、语音助手、对话机器人出身的，那转大模型算法是顺理成章。但如果你是 CS 应届生，或者从其他 IT 方向转过来，那"做应用"未必是最好的选择。

很多人有个误区，觉得大模型算法工程师就是调参、做预训练、做 SFT、做微调……
但实际上，这些活只占很小一部分，而且新人进去，90% 的概率不会直接让你碰模型调优。

更多可能是让你：

  • 配环境、搭链路
  • 清洗数据、分析数据
  • 做调研、写工具函数
  • 跑一些基础实验

等你把这些"杂活"干熟了，才可能接触线上业务。很多人干了好几年，还在打杂，根本摸不到核心。

所以，刚入行的新人，背景好的可以去大厂实习转正，背景一般的建议先去中小公司积累业务经验。

03 数据很重要！别看不起

很多人可能看不上"做数据的"，觉得学了那么多算法知识，最后去洗数据太屈才。

但我想说：做数据是很多转行同学更容易上岸的路。

条条大路通罗马，不是只有算法这一条。

目前国外大模型技术领先国内至少两年，虽然国内冒出一堆"大模型"，但真正能打的没几个。算法本身已经不是秘密，那差距在哪？一是数据，二是工程技巧。

就拿数据来说：

  • 通用大模型训练：数据从哪里采？质量怎么控？怎么过滤有毒信息？语言比例怎么定？怎么去重？评测集怎么建？
  • 垂直领域（金融、电商、法律等）：业务数据怎么来？数据不够怎么办？完全没有数据怎么办？怎么构建高质量的微调数据？

这些都是既考验体力又考验技术的活。能把数据问题解决好，模型就成功了一大半。

所以，有经验的数据工程师现在非常稀缺。

04 大模型平台是干啥的？

如果你之前是做工程的，或者对工程感兴趣，那我建议你选"平台方向"。

平台工程师就是做大模型的基础设施建设，目标是让模型训练得更快、更稳。具体干啥呢？

  • 计算层面：搞分布式计算、并行计算、高性能计算
  • 硬件层面：管理几百上千张卡，管 GPU 集群、混部集群，看利用率、机器健康
  • 平台层面：做 LLMOps，搭 Pipeline，把数据 IO、训练、预测、上线、监控串起来

这个方向在大模型时代需求稳中有升，很多公司都是从之前的深度学习平台团队转过来的，技术 gap 不大。

如果你对 AI 工程感兴趣，选这个方向很合适。

05 大模型部署是做啥的？

大模型部署工程师这两年特别火，为啥？
因为部署大模型太烧钱了。

模型一大，延迟就高，对算力、显存要求也高。老板们一方面要业务指标好看，另一方面也要控制成本。

所以，部署工程师的核心就是：让模型跑得更省、更快。

这个岗位主要有两个方向：

  • 云端部署：做推理加速平台、定制化加速、推理引擎，保证高并发下的延迟和吞吐
  • 端侧部署：在消费级 GPU/NPU 或边端设备上部署大模型，做模型小型化

这个岗位对工程能力、系统理解、硬件知识都有要求，虽然现在有各种推理框架降低了门槛，但仍然比较吃经验。

不建议新人直接冲这个方向，建议先从平台方向入手，再逐步转过来。

06 给新人的几点建议

最后总结几句，帮你理清思路：

  • 别只盯着微调、SFT、RLHF，作为学习可以，但别花太多精力
  • 想做应用，就聚焦到一个垂直领域，比如对话机器人、问答系统，或者金融/医疗/教育场景，做深做透
  • 多关注数据，Data Pipeline、高质量数据集的构建经验，这些对你未来工作最直接有用
  • 大模型不只有算法，工程同样重要。大公司拼的都是基建，好的平台是成功的基石

不管你选哪个方向，记住：先入场，再深耕。祝你顺利上岸！
